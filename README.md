# Presentación Diseño - Las Leyes del Diseño de Videojuegos

Una presentación interactiva sobre los fundamentos esenciales del diseño de videojuegos.

## Descripción

Este proyecto es una colección de 12 leyes del diseño de videojuegos, presentadas de manera interactiva y educativa. Cada ley está documentada en su propio archivo HTML con ejemplos, conceptos y aplicaciones prácticas.

## Estructura del Proyecto

```
presentacion leyes/
├── index.html                 # Página principal con navegación
├── PrimeraLey.html           # Primera Ley
├── SegundaLey.html           # Segunda Ley
├── terceraLey.html           # Tercera Ley
├── CuartaLey.html            # Cuarta Ley
├── QuintaLey.html            # Quinta Ley
├── SextaLey.html             # Sexta Ley
├── SeptimaLey.html           # Séptima Ley
├── OctavaLey.html            # Octava Ley
├── NovenaLey.html            # Novena Ley
├── DecimaLey.html            # Décima Ley
├── OnceLey.html              # Undécima Ley
├── DoceLey.html              # Duodécima Ley
└── README.md                 # Este archivo
```

## Características

✨ **Diseño Gaming:** Interfaz con estilo neón y efectos visuales modernos
📱 **Responsive:** Se adapta perfectamente a dispositivos móviles
🎯 **Navegación Intuitiva:** Acceso fácil a todas las leyes desde la página principal
⚔️ **Temática de Videojuegos:** Toda la presentación sigue una temática gaming coherente

## Cómo Usar

1. Abre `index.html` en tu navegador
2. Navega a través de las 12 leyes usando los botones de la página principal
3. Cada ley contiene información detallada, ejemplos y conceptos prácticos

## Requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Sin dependencias externas - HTML puro con CSS integrado
